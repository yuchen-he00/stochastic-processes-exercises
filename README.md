# 《计算机科学中的随机过程》配套习题

本仓库是《计算机科学中的随机过程》一书的配套开源习题库。

本书在上海交通大学电院人工智能专业 **AI2613《随机过程》** 课程讲义的基础上扩充而成，同时也是致远学院《概率论》、《高级算法选讲》等课程的参考教材。
