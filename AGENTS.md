# Project Skill Notes

## 通用注意事项

1. 行间公式中的左右大括号统一使用 `\left\lbrace` 和 `\right\rbrace`，不要使用其他写法（例如 `\left\{`、`\right\}`）。
2. 本文件用于记录项目范围内的通用写作与排版约定；新增规则时请保持表述简洁、可执行。
3. 其他公式中使用的宏，统一参考 `docs/javascripts/mathjax.js` 中的定义，不要自行引入不在该文件中的宏写法。
4. 提示（Hint）统一使用可收放语法 `??? hint "提示"`（默认收起），用于给思路引导但不直接给完整证明；视觉上使用与项目既有 hint 一致的轻字体、浅背景框样式。
5. 额外引用且不要求学生证明的定理/引理，统一使用非收放语法 `!!! theorem "定理（可直接使用）"`（引理可改标题为“引理（可直接使用）”）；其视觉样式应与 hint 保持同一背景色与弱化风格，用于与正文区分但不做强强调。
6. 在多行数学环境（如 `\begin{aligned}`、`\begin{cases}` 等）中，**不要使用 `\\` 换行**，统一使用 `\cr`。原因是 Markdown 解析器可能在 `\\` 到达 MathJax 之前错误地转义或吞掉反斜杠。
